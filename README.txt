Run the commands
	make -- to compile.
	./shell -- to run the shell

shell.c file:
	It has the main function which sets the prompt each time.

operate.c
	It has the functions to read the command , parse it and execute it.

extern_com.c
	It has the functions for built-in commands

operate.h 
	It is a header file for all the 3 .c files
